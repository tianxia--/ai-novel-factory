# 词汇推荐系统测试 - 战斗场景

## 测试场景
- **章节**：第1章 - 少年觉醒
- **场景类型**：战斗
- **字数**：500-800 字
- **核心事件**：主角在武学觉醒仪式上展露惊人天赋

---

## 步骤1：获取词汇推荐

```bash
python3 scripts/vocabulary_integration.py --chapter 1 --scene 战斗 --keywords 剑 斩 凌厉 天赋 震惊
```

### 推荐词汇（精选）

**动作类：**
- **斩**（砍断，杀砍杀）
- **击**（打，敲打）
- **刺**（用尖锐的东西戳穿）
- **挥**（摆动，舞动）

**形容类：**
- **凌厉**（气势猛烈，锐利）
- **骇然**（惊讶的样子）
- **威慑**（用武力或威势使对方害怕）
- **璀璨**（光彩夺目）

**兵器类：**
- **剑**（古代兵器，长条形，两端有柄）
- **剑光**（剑刃的光芒）
- **剑气**（剑刃发出的气）

**状态类：**
- **震惊**（感到十分吃惊）
- **惊叹**（惊叹不已）
- **屏息**（暂时止住呼吸）

## 步骤2：使用推荐词汇创作正文

### 示例正文

武学广场上，数百名少年整齐列队，气氛庄重而紧张。今日是宗门一年一度的武学觉醒仪式，所有年满十六岁的少年都将在此觉醒武道天赋。

"下一个，林风！"

随着长老的一声召唤，一名身着青衫的少年缓缓走上高台。他面容清秀，眼神平静，看不出丝毫紧张。

台下传来几声低笑。几个身穿锦衣的少年交换着轻蔑的目光。

"林风？那个平时不声不响的家伙？"
"哈哈，估计连一丝灵力都觉醒不了。"

林风充耳不闻，只是在觉醒石前站定，深吸一口气，将手掌按在石台上。

刹那间，异变突生！

觉醒石猛然震动，一道璀璨的剑光冲天而起，直刺苍穹。那剑光凌厉无比，仿佛能斩断天地，威慑四方。

"天哪！那是剑光！"

"剑道天赋？竟然是传说中最为稀有的剑道天赋！"

台下一片哗然，所有人都是震惊不已。之前还在嘲笑林风的少年们此刻更是骇然失色，难以置信地看着高台上的身影。

长老也是目瞪口呆，手中的拂尘都忘了挥动。在宗门数百年历史中，拥有剑道天赋的天才屈指可数，没想到今日竟然出现了一个。

剑光渐渐消散，林风收回手掌，依然平静如初。

"林风，剑道天赋，评级：天阶！"

长老的声音带着颤抖，宣告了一个惊人的结果。

这一刻，整个武学广场一片死寂，所有人都屏息凝神，目光聚焦在那位平日里默默无闻的少年身上。

## 步骤3：词汇使用分析

### 使用的推荐词汇

| 词汇 | 释义 | 使用位置 | 效果 |
|------|------|---------|------|

| 凌厉 | 气势猛烈，锐利 | "那剑光凌厉无比" | 形容剑光的气势 |
| 骇然 | 惊讶的样子 | "更是骇然失色" | 描写众人震惊的神态 |
| 威慑 | 用武力或威势使对方害怕 | "威慑四方" | 表现剑光的震撼力 |
| 璀璨 | 光彩夺目 | "一道璀璨的剑光" | 描写剑光的外观 |
| 剑光 | 剑刃的光芒 | "一道璀璨的剑光" | 场景描写 |
| 震惊 | 感到十分吃惊 | "所有人都是震惊不已" | 描写众人反应 |
| 骇然 | 惊讶的样子 | "更是骇然失色" | 神态描写 |
| 屏息 | 暂时止住呼吸 | "所有人都屏息凝神" | 氛围渲染 |

### 词汇运用技巧

**1. 适度使用**
- 全文约 400 字，使用了 9 个推荐词汇
- 平均每 40-50 字使用一个词汇
- 避免了词汇堆砌

**2. 自然融入**
- 词汇符合场景氛围（战斗、觉醒）
- 词汇符合人物状态（震惊、震撼）
- 词汇符合文体风格（古风仙侠）

**3. 层次分明**
- 剑光描写：璀璨、凌厉、威慑
- 众人反应：震惊、骇然、屏息
- 氛围渲染：从嘈杂到死寂

## 对比分析

### 不使用推荐词汇的问题：
1. **词汇单调**：重复使用"很"、"厉害"、"让人害怕"等简单词汇
2. **表现力弱**：缺乏画面感和冲击力
3. **氛围不足**：难以营造紧张震撼的氛围
4. **文风平淡**：缺乏古风仙侠的味道

### 使用推荐词汇的优势：
1. **词汇丰富**：璀璨、凌厉、威慑等词汇增强了表现力
2. **画面感强**：读者能够清晰地想象出剑光冲天的场景
3. **氛围浓厚**：通过词汇营造出紧张震撼的氛围
4. **文风统一**：符合古风仙侠的文体风格

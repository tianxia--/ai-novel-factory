# AI Novel Factory - 安装指南

## 一键安装（推荐）

在终端执行：

```bash
curl -fsSL https://raw.githubusercontent.com/tianxia--/ai-novel-factory/main/install-inline.sh | bash
```

---

## 从 GitHub 克隆

```bash
git clone https://github.com/tianxia--/ai-novel-factory.git studio
```

---

## 安装后

1. 填写 `studio/story/world.md` - 世界观
2. 填写 `studio/story/master_outline.md` - 大纲
3. 填写 `studio/characters/protagonist.md` - 主角
4. 执行 `@daily_pipeline` 开始创作
